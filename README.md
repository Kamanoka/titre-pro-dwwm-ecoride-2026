# EcoRide - Plateforme de covoiturage écologique

Projet ECF - Titre Professionnel Développeur Web et Web Mobile (2026)

## Description
EcoRide est une plateforme de covoiturage favorisant les trajets écologiques (priorité aux véhicules électriques) et économiques.

## Stack technique
- Front : HTML5, CSS, JavaScript
- Back : PHP 8 + PDO
- BDD relationnelle : MySQL
- BDD NoSQL : MongoDB (avis et préférences libres)
- Déploiement : en cours

## Installation locale
1. Cloner le repo
2. Créer la base MySQL `ecoride`
3. Importer sql/create-tables.sql puis seed-data.sql
4. Lancer avec `php -S localhost:8000 -t public`
